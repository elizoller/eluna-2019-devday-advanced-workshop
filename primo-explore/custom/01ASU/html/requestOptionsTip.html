Request Options Help

Request ASU Copy
If copy is available, it will be delivered in 24-72 hours.  If copy is not available, it will be recalled from another user and delivered within 2 weeks.

Request External Copy
If ASU copy is not available, you may request it from an external library (ILL).  External copies may arrive quicker but have shorter loan periods with no renewals.  Estimated delivery is 5-7 business days.

Request Chapter Scan
Fair use allows for book chapters to be scanned (one chapter per request), not to exceed 10% of the book.  Scans will be delivered via PDF in 24-72 hours.

Request Article Scan
Fair use allows for journal articles to be scanned (one article per request), not to exceed 10% of the journal issue.  Scans will be delivered via PDF in 24-72 hours.

Request a different item
If ASU copy is available, it will be delivered in 24-72 hours.  If copy is not available, it will be recalled from another user and delivered within 2 weeks.

Request a scan of a different item
Fair use allows for journal articles to be scanned (one article per request), not to exceed 10% of the journal issue.  Scans will be delivered via PDF in 24-72 hours.
